Stock Market Dashboard

A modern stock market dashboard built with Next.js 14 (App Router), Prisma, Finnhub API, and Recharts.
It provides real-time stock data, earnings charts, watchlist, and portfolio management.

🚀 Features
    📊 Live stock price tracking
    📈 Earnings vs Estimates charts
    ⭐ Watchlist (add/remove toggle)
    💼 Portfolio management
    🏢 Company details (market cap, industry, IPO, etc.)
    🔍 Stock search & selection
    🌙 Dark/Light mode support
    ⚡ Fast API routes with Next.js App Router

Tech Stack
    Frontend: Next.js 14, React, Tailwind CSS
    Backend: Next.js API Routes
    Database: Prisma + MongoDB / PostgreSQL
    Charts: Recharts
    API: Finnhub Stock API
    Auth: NextAuth (session-based)
